# vue-template
tired of using vue-cli to create a vue prj everytime

每次新建一个vue项目都要重新使用vue-cli构建node_modules。。

然而这些玩意都是重复的，好蠢，干脆把src里的内容写成一个固定的分支main，每次开新项目就把旧项目作为某分支存起来，然后从main拉取原始代码开新分支得了
